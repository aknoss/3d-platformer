# 3D Platformer

A 3D platformer built with C++, SDL3, and OpenGL 3.3 Core. Features flat-shaded colored geometry, third-person camera, coin collecting, and AABB collision.

## Building

**Requirements:** g++, SDL3 (system-wide via pkg-config), OpenGL

```bash
make          # builds ./game
make clean    # removes binary and .o files
```

## Run

```bash
./game
```
